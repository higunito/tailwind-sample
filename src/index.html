スキルスタック

使用可能な技術スタック

HTML

HTMLはWebページの構造を作成するための言語です。HTMLを使用することで、Webページの見た目や機能を作成することができます。

CSS

CSSはWebページのデザインを作成するための言語です。CSSを使用することで、Webページの見た目を整えることができます。

TailwindCSS

TailwindCSSはCSSフレームワークです。TailwindCSSを使用することで、Webページのデ ザインを簡単に作成することができます。

JavaScript

JavaScriptはWebページの動的な機能を作成するための言語です。JavaScriptを使用することで、Webページに動きを付けることができます。

TypeScript

TypeScriptはJavaScriptのスーパーセットです。TypeScriptを使用することで、JavaScriptのコードをより安全に書くことができます。

React

ReactはFacebookが開発したJavaScriptライブラリです。Reactを使用することで、Webページのコンポーネントを作成することができます。

Next.js

Next.jsはReactのフレームワークです。Next.jsを使用することで、Reactアプリケーションを簡単に作成することができます。

Go

GoはGoogleが開発したプログラミング言語です。Goを使用することで、Webアプリケーションを高速に作成することができます。

AWS

AWSはAmazonが提供するクラウドサービスです。AWSを使用することで、Webアプリケーションを簡単にデプロイすることができます。

Terraform

Terraformはインフラストラクチャをコードで管理するためのIaCツールです。Terraformを使用することで、インフラストラクチャを安全に管理することができます。

Docker

Dockerはコンテナ仮想化を行うためのツールです。

Kubernetes

Kubernetesはコンテナオーケストレーションツールです。Kubernetesを使用することで、コンテナを簡単に管理することができます。
